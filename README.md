# Task-Flow-Manager-Odana
Proyecto CRUD donde podras manajar tus proyecto.


Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Node.js
- Express
- SQLite

Funcionalidades

- Inicio de sesión simple
- Crear proyectos
- Listar proyectos
- Editar proyectos
- Eliminar proyectos
- Agregar subtareas dinámicas
- Cambiar estado del proyecto
- Filtrar proyectos por estado

 Estados del proyecto

- To Do
- In Progress
- Completed

 Cómo ejecutar

Dentro de la carpeta `app`:


npm install
npm start
